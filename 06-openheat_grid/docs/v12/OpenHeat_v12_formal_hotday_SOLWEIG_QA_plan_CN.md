# OpenHeat v1.2-beta formal-hot-day SOLWEIG QA plan

**Document date:** 2026-05-25
**Stage:** v1.2-beta.3 planning
**Status:** plan before local QGIS/SOLWEIG execution

## 0. Purpose

This QA tests whether the accepted Core-8 target and typology interpretation remain coherent under formal-hot-day forcing.

It is not a scale run and not a hazard/risk product.

## 1. Core question

```text
Do Core-8 target roles remain directionally stable when forcing changes from v10-epsilon forcing to formal-hot-day forcing?
```

## 2. Minimal smoke matrix

Run:

```text
5 cells x 2 hours x 2 scenarios = 20 SOLWEIG runs
```

Cells:

| cell_id | reason |
|---|---|
| TP_0986 | high-exposure null-control; overhead delta should remain near zero |
| TP_0565 | road-edge hot anchor; overhead delta should remain near zero |
| TP_0542 | mapped pedestrian-overhead shade case; distribution response should remain interpretable |
| TP_0059 | hardscape mean-vs-p90 diagnostic |
| TP_0835 | wooded low-radiative lower anchor |

Hours:

```text

```


## 5. Acceptance criteria

### TP_0986

Expected:

```text
p90 remains high;
overhead p90 delta remains near zero;
no mapped-overhead response should appear.
```

Fail condition:

```text
overhead p90 delta becomes large without geometry explanation.
```

### TP_0565

Expected:

```text
p90 remains high;
overhead p90 delta remains near zero.
```

Fail condition:

```text
hot-anchor role disappears or large overhead response appears unexpectedly.
```

### TP_0542

Expected:

```text
overhead affects p90 or distribution in a shade-consistent way;
not necessarily identical magnitude to v10-epsilon forcing.
```

Fail condition:

```text
mapped overhead becomes directionally non-interpretable after checking forcing/time/geometry.
```

### TP_0059

Expected:

```text
mean-vs-p90 distinction remains useful;
mean may move more than p90.
```

Fail condition:

```text
p90 collapses unexpectedly, suggesting mask/overhead overcoverage issue.
```

### TP_0835

Expected:

```text
remains low-radiative lower anchor;
mean, p90, max stay relatively close.
```

Fail condition:

```text
becomes high-radiative without vegetation/forcing explanation.
```
